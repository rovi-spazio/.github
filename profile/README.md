# 🌿 Rovi Spazio Creativo ETS

Sviluppiamo l'infrastruttura digitale per l'hub di comunità di Roviano. Uniamo rigore tecnico e approcci snelli ed essenziali per creare soluzioni scalabili a supporto della rigenerazione territoriale della Valle dell'Aniene.

## 🚀 La Piattaforma Coworking
Il nostro prodotto principale è un'applicazione distribuita che gestisce l'intera esperienza utente all'interno degli spazi Rovi, strutturata nei seguenti moduli:

- **`coworking_app`**: Il core system e l'interfaccia utente per la gestione di spazi e prenotazioni (Frontend JS / Backend Python).
- **`rovi-gale1-iam-platform`**: Modulo centralizzato per l'Identity & Access Management, che garantisce la sicurezza e la gestione dei profili.
- **`rovi-be-payments-service`**: Microservizio indipendente (Java) dedicato all'elaborazione sicura delle transazioni e agli abbonamenti.

## 🌐 Ecosistema Web & Admin
- **`rovispaziocreativo`**: Landing page istituzionale e vetrina dei servizi. Sviluppata in vanilla HTML/CSS/JS per la massima leggerezza.
- **`rovi-gestionale`**: Pannello di amministrazione (TypeScript/HTML) per il controllo interno e l'analisi dei dati.
- **`rovi-social-canvas`**: Progetto open source e R&D basato su Python.

## 🛠 Tech Stack
Privilegiamo framework essenziali e veloci:
React, Vite, Tailwind CSS • Python • Java (Microservizi) • Firebase • Docker
